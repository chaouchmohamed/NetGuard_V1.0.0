NetGuard is a real-time network attack detection system that uses machine learning (Isolation Forest) to identify anomalous network traffic patterns. It features a beautiful cyberpunk-themed dashboard with live traffic visualization, threat alerts, and interactive charts.

## ✨ Key Features

- **Real-time Traffic Analysis**: Continuous monitoring of network traffic with ML-based anomaly detection
- **Interactive Dashboard**: Cyberpunk-themed UI with live updates via WebSocket
- **Multiple Attack Detection**: Identifies DDoS, port scans, SYN floods, brute force attacks, and data exfiltration
- **Live Visualizations**: Real-time charts, network maps, and threat alerts
- **Historical Data**: SQLite database for storing traffic logs and alerts
- **RESTful API**: Comprehensive API for data access and system control
- **Docker Support**: Easy deployment with Docker Compose
